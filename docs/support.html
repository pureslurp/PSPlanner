Frequently Asked Questions

How do I create a task?
Tap the "+" button in the bottom right corner. Select whether you want to create a Daily, Weekly, or Monthly task, add a title, optionally set a category and deadline, then tap "Save".
How do tasks carry forward?
Incomplete tasks automatically carry forward. Daily tasks that aren't completed will appear the next day, weekly tasks will appear in the next week, and monthly tasks will appear in the next month.
Can I set deadlines on tasks?
Yes! When creating or editing a task, toggle "Set Deadline" and choose a date (for weekly/monthly tasks) or time (for daily tasks). Tasks with deadlines will automatically appear in your daily view when the deadline approaches.
How do categories work?
Categories help you organize tasks by type (e.g., Work, Personal, Errands). You can assign a category when creating or editing a task. Tap the menu icon (⋮) and select "Manage Categories" to create, edit, or delete categories.
How do notifications work?
For tasks with deadlines, you'll receive reminders 1 hour before (daily tasks), 1 day before (weekly tasks), or 3 days before (monthly tasks). For tasks without deadlines, you'll receive recurring reminders (daily at 8pm, weekly on Sundays, monthly on the 23rd). You can manage notifications in Settings.
Can I edit or delete tasks?
Yes! Tap any task to edit it. To delete a task, swipe left on it in the list and tap the delete button.
How do I mark a task as complete?
Tap the circle icon next to any task to mark it as complete. Completed tasks will appear in the "Completed" section and can be deleted if desired.

Getting Started

If you're new to Gotta, here's a quick guide:

  1. Complete the onboarding flow to learn about the app's features
  2. Create your first task by tapping the "+" button
  3. Organize tasks into categories for better visibility
  4. Set deadlines for important tasks
  5. Enable notifications in Settings to get reminders

Features Overview

Daily Tasks

Tasks that need to be done today. These are high-priority items that should be completed daily.

Weekly Tasks

Tasks that can be completed over the course of a week. Perfect for planning weekly goals and objectives.

Monthly Tasks

Longer-term tasks and goals that span a month. Great for tracking monthly milestones and objectives.

Deadline Promotion

Weekly and monthly tasks with deadlines will automatically appear in your daily view when the deadline approaches, so you never miss important due dates.

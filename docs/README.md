# GitHub Pages Site for Gotta

This directory contains the GitHub Pages site for the Gotta iOS app.

## Pages

- `index.html` - Main landing/marketing page
- `support.html` - Support page with FAQs and contact information

## Setup Instructions

1. **Enable GitHub Pages:**
   - Go to your GitHub repository settings
   - Scroll to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Select branch: `main` (or `master`)
   - Select folder: `/docs`
   - Click "Save"

2. **Your site will be available at:**
   - `https://[your-username].github.io/[repository-name]/`
   - Support URL: `https://[your-username].github.io/[repository-name]/support`

3. **For App Store Connect:**
   - **Support URL:** `https://[your-username].github.io/[repository-name]/support`
   - **Marketing URL (optional):** `https://[your-username].github.io/[repository-name]/` or leave blank

## Customization

- Update the email address in `support.html` (search for "support@example.com")
- Modify content as needed for your app
- Add screenshots if desired

## Notes

- Make sure to commit these files to your repository
- GitHub Pages may take a few minutes to deploy after pushing changes
